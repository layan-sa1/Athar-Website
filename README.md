# 🌱 Athar - Eco-Friendly E-Commerce Platform

Athar is a full-stack web application developed as a **Web Systems course project**. It's dedicated to promoting sustainable and eco-friendly products while demonstrating comprehensive full-stack web development concepts.

## ✨ Features

- **Product Catalog** — Browse a curated collection of eco-friendly products
- **Shopping Cart** — Add products and manage your cart seamlessly
- **Secure Checkout** — Complete your purchase with confidence
- **Admin Panel** — Manage products and inventory (with authentication)
- **Responsive Design** — Optimized for desktop, tablet, and mobile devices
- **Secure Database** — MySQL-powered backend for reliable data management

## 🛠️ Technologies Used

- **Frontend:** HTML5, CSS3, JavaScript
- **Backend:** PHP
- **Database:** MySQL
- **Architecture:** Full-Stack MVC Pattern
